Description:

Asus TUF Gaming F15 (2022) was first announced at CES 2022 with the latest 12th Gen Intel H-series mobile processors. The FX507ZM, in particular, is powered by Core i7-12700H CPU with 14 cores (6 P-cores + 8 E-cores). It is paired with RTX 3060 graphics with a total TGP of 140W (115 + 25W Dynamic Boost). The laptop features a 15.6-inch WQHDscreen with a 165Hz refresh rate and 100% sRGB gamut. It also supports Adaptive-Sync, MUX Switch, and Optimus. Memory options include up to 32GB DDR5 RAM and 1TB SSD.

Specifications:

  • Processor: Intel i7-12700H
  • RAM: 16GB
  • Storage: 1TB SSD
  • Graphics: NVIDIA® GeForce RTX™ 3060 Laptop GPU
  • Display: 15.6" FHD, 144Hz
  • Battery: 90Wh 4-cell Lithium Ion
  • Ports: HDMI, USB-C, 3x USB-A, LAN
